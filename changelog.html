Changelog

v1.2  |  June 17 2025  |  Improvements & New Functionality
  • User logging API – new /v1/user-logs endpoints provide fine-grained audit trails for web & mobile access, including source (web/mobile), action (view/add/edit/delete) and camera/group counts. (see Camcloud blog)
  • Added helper examples in the docs showing how to query logs for a single user or an entire account, plus sample dashboards using Klipfolio.
v1.1  |  January 12 2025  |  New Functionality
  • Live View streaming API – introduced HLS endpoints (/v1/cameras/{camera_id}/stream) for low-latency viewing across browsers and mobile apps.
  • Media download API – new /v1/media/{media_id}/download route lets clients export MP4 clips or JPEG snapshots on demand. Supports time-range parameters and direct-to-S3 transfer.
  • Updated authorization flow to support long-lived tokens for continuous playback & downloads.
v1.0  |  December 02 2024  |  Initial Release

No additional notes.
